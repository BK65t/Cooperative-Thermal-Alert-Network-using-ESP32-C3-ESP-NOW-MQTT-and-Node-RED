## About the Project  
The Cooperative Thermal Alert Network (CTAN) is a hybrid distributed IoT network designed to achieve:

- Ultra-low energy consumption  
- Local peer-to-peer coordination  
- Cloud integration through MQTT  
- Near-instant alert propagation  
- Secure AES-128 encrypted communication
